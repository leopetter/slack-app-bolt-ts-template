# Simple Slack App using Bolt and Typescript

Bolt is a framework to build Slack apps in a flash with the latest platform features. To better understand Bolt-JS, please read the [getting started guide](https://slack.dev/bolt-js/tutorial/getting-started).

This Repo is used as a template to start building Slack Apps using Typescript instead of Javascript. 

## Setup
 ```bash
npm install

# lint and automatically fix issues
npm run lint-and-fix

# start Bolt app
npm run start
```
